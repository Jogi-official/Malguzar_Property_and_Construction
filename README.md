# Malgulzar Property & Construction Website

A modern real estate and construction website built using **HTML + Tailwind CSS** for showcasing verified properties, land deals, and construction services in Dehradun.

---

## ✨ Features

- Modern responsive UI
- Hero landing section
- About Us section
- Services section
- Featured property cards
- Google Maps integration
- Instagram Reel integration
- WhatsApp contact button
- Mobile-first responsive design
- Custom favicon/logo
- Clean Tailwind CSS styling

---

## 📸 Sections Included

### 🏠 Hero Section
- Trust badge
- Strong CTA
- Promotional property image

### 👨‍💼 About Section
- Company overview
- Experience highlights
- Customer statistics

### 🏗 Services Section
- Land Buying & Selling
- Construction Services
- Property Consultation

### 📍 Featured Properties
- Property cards
- Property highlights
- Google Maps location
- Instagram reel/video support

### 📞 Contact Section
- Direct WhatsApp enquiry button
